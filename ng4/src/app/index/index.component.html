Want to write high performance code?

Circle Academy helps you write better code. Faster. Leaner. And Elegantly.

Learn any of these following languages:

C/++

The gold standard for systems level programming. Pretty much everything is built on ANSI C or C++ these days. Learn how to manage memory. Data structures and write high performance code.

Java/Scala

The JVM ecosystem is robust and powerful. Learn how to use the most powerful tools in Java. Learn data structures and how to write performant code.

Golang

A staple language for writing high performance multi threaded programs. Golang is an excellent general purpose language.

Elixir

Erlang/Elixir is the ultimate networking language. Write high performance network based application in Elixir. Designed specifically for web.

SQL

MySQL, Postgres and SQLite are the gold standard for data storage and high throughput concurrent read/writes. Learn how to traverse complex queries and other things.

Node.js

Javascript and Node.js are the defacto standard for front end based applications. Learn to write React/Redux and Angular 4 web apps.

Swift

Swift 3 is the gold standard for iOS applications. Build clean scalable iOS apps.

Python

The Python and PyPi toolkit are unbeatable. Learn how to write Python and utilize the Python ecosystem.
